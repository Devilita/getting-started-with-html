Key Concepts

The Web
The Web consists of many servers that are connected to each other and talk using the Hyper-text transfer protocol (http). Browsers fetch the Hyper-text markup language document (HTML doc) and render it for display. Those HTML documents can contain plain text, images, links, videos, and many more items.

HTML
Hyper-text markup language (HTML) is used to format pages. It can do so by using tag elements. An element in its most basic form consists of an opening tag, some content, and a closing tag. For example <tag>content</tag>. One needs to be careful as the computer will always do as told - even if there are mistakes in the code, which it will not notice.
